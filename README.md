<h1>PROGRAMA TABLERO</h1>

3 automatas compiten por llegar a su casilla final primero.

Cada automata al empezar crea todas las posibles rutas a la victoria.

Se puede observar la ruta escogida para jugar, si cambia de ruta, el grafo de posibilidades por automata.
